# Stepik Selenium Language Tests

Небольшой учебный проект для курса **“Автоматизация тестирования с помощью Selenium и Python”** на Stepik.

Цель проекта — показать, как запускать автотесты Selenium с различными языками интерфейса, передавая язык через параметр командной строки PyTest.

---

## 📌 Описание

Интернет-магазин должен корректно работать для пользователей из разных стран.  
В этом проекте реализована возможность запуска автотестов для разных языков интерфейса.

Тест проверяет, что на странице товара присутствует кнопка **добавления в корзину**.

Язык интерфейса браузера задаётся параметром:

```
--language
```

---

## 📂 Структура проекта

```
stepik_test
│
├── conftest.py
└── test_items.py
```

### `conftest.py`

- добавляет параметр `--language` в PyTest
- запускает браузер Chrome с выбранным языком
- предоставляет фикстуру `browser`

### `test_items.py`

- открывает страницу товара
- проверяет наличие кнопки **Add to basket**

---

## 🚀 Запуск тестов

Пример запуска теста с испанским языком:

```bash
pytest --language=es test_items.py
```

---

## 🌍 Проверка разных языков

Можно запускать тесты для разных языков интерфейса.

Например:

```bash
pytest --language=fr test_items.py
```

После открытия страницы кнопка добавления в корзину будет отображаться как:

```
Ajouter au panier
```
